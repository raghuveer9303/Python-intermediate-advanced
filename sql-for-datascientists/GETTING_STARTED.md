## Quick Start Guide

### 1. Prerequisites
- Basic SQL knowledge (SELECT, FROM, WHERE)
- SQL environment (PostgreSQL, MySQL, or online)
- Understanding of relational databases

### 2. Setup

#### Option 1: PostgreSQL (Recommended)
```bash
# Install PostgreSQL
# Create database
createdb ds_practice

# Connect
psql ds_practice
```

#### Option 2: Online SQL Practice
- **SQLFiddle.com** - Quick testing
- **DB-Fiddle.com** - Multiple databases
- **LeetCode SQL** - Interview practice

### 3. Learning Order

**Week 1: Foundation**
1. Start with `01-joins/`
   - Run `sample_data.sql` to create tables
   - Complete `exercises.sql`
   - Review `solutions.sql`
2. Practice JOIN problems on LeetCode

**Week 2: Advanced**
1. Move to `02-window-functions/`
2. Practice window function problems
3. Learn aggregations and subqueries

**Week 3-4: Mastery**
1. Complex queries and patterns
2. Performance optimization
3. Real-world scenarios
