## üìã Learning Path Overview

### Phase 1: Foundation (Week 1)
1. **SELECT & WHERE** - Basic queries
2. **JOINs** - Combining tables (most important!)
3. **Aggregations** - GROUP BY, COUNT, SUM, AVG

### Phase 2: Advanced Queries (Week 2)
4. **Window Functions** - ROW_NUMBER, RANK, LAG, LEAD
5. **Subqueries & CTEs** - Complex logic
6. **Date Functions** - Time series analysis

### Phase 3: Data Science Specific (Week 3)
7. **Statistical Functions** - Percentiles, standard deviation
8. **Data Cleaning** - Handling NULLs, duplicates
9. **Pivot & Unpivot** - Reshaping data

### Phase 4: Optimization (Week 4)
10. **Indexes & Performance** - Query optimization
11. **Complex Patterns** - Real-world scenarios
12. **Common DS Problems** - Interview patterns
